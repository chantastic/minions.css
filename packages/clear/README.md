# minions.clear
minions clear classes

## Module
name: `minions.minions.clear`  
version: `0.2.0`  
main/style: `clear.css`  

## Installation
npm:
```bash
npm install minions.clear
```

browser:
```html
<link rel="stylesheet" href="https://unpkg.com/minions.clear" />
```

## Code
```css
/*! minions.css */
.c-n{clear:none}
.c-l{clear:left}
.c-r{clear:right}
.c-b{clear:both}

```
