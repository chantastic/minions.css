# minions.clear
minions clear classes

## Module
name: `minions.minions.clear`  
version: `0.0.4-alpha-7`  
main/style: `clear.css`  

## Installation
npm:
```bash
npm install minions.clear@0.0.4-alpha-7
```

browser:
```html
<link rel="stylesheet" href="https://unpkg.com/minions.clear@0.0.4-alpha-7" />
```

## Code
```css
/*! minions.css */
.c-n{clear:none}
.c-l{clear:left}
.c-r{clear:right}
.c-b{clear:both}

```
