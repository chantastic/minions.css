# minions.height
minions height classes

## Module
name: `minions.minions.height`  
version: `0.3.0`  
main/style: `height.css`  

## Installation
npm:
```bash
npm install minions.height
```

browser:
```html
<link rel="stylesheet" href="https://unpkg.com/minions.height" />
```
