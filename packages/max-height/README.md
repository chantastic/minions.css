# minions.max-height
minions max-height classes

## Module
name: `minions.minions.max-height`  
version: `0.3.0`  
main/style: `max-height.css`  

## Installation
npm:
```bash
npm install minions.max-height
```

browser:
```html
<link rel="stylesheet" href="https://unpkg.com/minions.max-height" />
```
