# minions.flex-basis
minions flex-basis classes

## Module
name: `minions.minions.flex-basis`  
version: `0.3.1`  
main/style: `flex-basis.css`  

## Installation
npm:
```bash
npm install minions.flex-basis
```

browser:
```html
<link rel="stylesheet" href="https://unpkg.com/minions.flex-basis" />
```

## Code
```css
/*! minions.css */
.fb-a{flex-basis:auto}
.fb-0{flex-basis:0}
.fb-0\%{flex-basis:0%}

```
