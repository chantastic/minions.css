# minions.float
minions float classes

## Module
name: `minions.minions.float`  
version: `0.0.4-alpha-4`  
main/style: `float.css`  

## Installation
npm:
```bash
npm install minions.float@0.0.4-alpha-4
```

browser:
```html
<link rel="stylesheet" href="https://unpkg.com/minions.float@0.0.4-alpha-4" />
```

## Code
```css
/*! minions.css */
.f-l{float:left}
.f-n{float:none}
.f-r{float:right}

```
