# minions.float
minions float classes

## Module
name: `minions.minions.float`  
version: `0.2.0`  
main/style: `float.css`  

## Installation
npm:
```bash
npm install minions.float
```

browser:
```html
<link rel="stylesheet" href="https://unpkg.com/minions.float" />
```

## Code
```css
/*! minions.css */
.f-l{float:left}
.f-n{float:none}
.f-r{float:right}

```
