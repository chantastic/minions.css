# minions.resize
minions resize classes

## Module
name: `minions.minions.resize`  
main/style: `resize.css`  

## Installation
npm:
```bash
npm install minions.resize
```

browser:
```html
<link rel="stylesheet" href="https://unpkg.com/minions.resize" />
```

## Code
```css
/*! minions.css */
.r-n{resize:none}
.r-b{resize:both}
.r-h{resize:horizontal}
.r-v{resize:vertical}
.r-b{resize:block}
.r-i{resize:inline}

```
