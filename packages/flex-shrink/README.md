# minions.flex-shrink
minions flex-shrink classes

## Module
name: `minions.minions.flex-shrink`  
version: `0.3.0`  
main/style: `flex-shrink.css`  

## Installation
npm:
```bash
npm install minions.flex-shrink
```

browser:
```html
<link rel="stylesheet" href="https://unpkg.com/minions.flex-shrink" />
```

## Code
```css
/*! minions.css */
.fs-0{flex-shrink:0}
.fs-1{flex-shrink:1}
.fs-2{flex-shrink:2}
.fs-3{flex-shrink:3}
.fs-4{flex-shrink:4}
.fs-5{flex-shrink:5}
.fs-6{flex-shrink:6}

```
