# minions.width
minions width classes

## Module
name: `minions.minions.width`  
version: `0.3.0`  
main/style: `width.css`  

## Installation
npm:
```bash
npm install minions.width
```

browser:
```html
<link rel="stylesheet" href="https://unpkg.com/minions.width" />
```

## Code

/*! minions.css */

.w-nc{width:min-content}
