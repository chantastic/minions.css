# minions.flex-grow
minions flex-grow classes

## Module
name: `minions.minions.flex-grow`  
version: `0.0.4-alpha-15`  
main/style: `flex-grow.css`  

## Installation
npm:
```bash
npm install minions.flex-grow
```

browser:
```html
<link rel="stylesheet" href="https://unpkg.com/minions.flex-grow" />
```

## Code
```css
/*! minions.css */
.fg-1{flex-grow:1}
.fg-2{flex-grow:2}
.fg-3{flex-grow:3}
.fg-4{flex-grow:4}
.fg-5{flex-grow:5}
.fg-6{flex-grow:6}

```
