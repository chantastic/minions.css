# minions.padding
minions padding classes

## Module
name: `minions.minions.padding`  
version: `0.0.4-alpah-13`  
main/style: `padding.css`  

## Installation
npm:
```bash
npm install minions.padding@0.0.4-alpah-13
```

browser:
```html
<link rel="stylesheet" href="https://unpkg.com/minions.padding@0.0.4-alpah-13" />
```

## Code
```css
/*! minions.css */
.p-0{padding:0}
.p-\.25e{padding:.25em}
.p-\.5e{padding:.5em}
.p-1e{padding:1em}
.p-2e{padding:2em}
.p-3e{padding:3em}
.p-4e{padding:4em}
.p-\.25r{padding:.25rem}
.p-\.5r{padding:.5rem}
.p-1r{padding:1rem}
.p-2r{padding:2rem}
.p-3r{padding:3rem}
.p-4r{padding:4rem}

```
