# minions.flex
minions flex classes

## Module
name: `minions.minions.flex`  
version: `0.0.4-alpha-3`  
main/style: `flex.css`  

## Installation
npm:
```bash
npm install minions.flex@0.0.4-alpha-3
```

browser:
```html
<link rel="stylesheet" href="https://unpkg.com/minions.flex@0.0.4-alpha-3" />
```

## Code
```css
/*! minions.css */
.f-1{flex:1}
.f-2{flex:2}
.f-3{flex:3}
.f-4{flex:4}
.f-5{flex:5}
.f-6{flex:6}

```
