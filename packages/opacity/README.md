# minions.opacity
minions opacity classes

## Module
name: `minions.minions.opacity`  
version: `0.0.4-alpha-4`  
main/style: `opacity.css`  

## Installation
npm:
```bash
npm install minions.opacity@0.0.4-alpha-4
```

browser:
```html
<link rel="stylesheet" href="https://unpkg.com/minions.opacity@0.0.4-alpha-4" />
```

## Code
```css
/*! minions.css */
.o-1{opacity:1}
.o-\.9{opacity:.9}
.o-\.8{opacity:.8}
.o-\.7{opacity:.7}
.o-\.6{opacity:.6}
.o-\.5{opacity:.5}
.o-\.4{opacity:.4}
.o-\.3{opacity:.3}
.o-\.2{opacity:.2}
.o-\.1{opacity:.1}
.o-0{opacity:0}

```
