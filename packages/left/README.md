# minions.left
minions left classes

## Module
name: `minions.minions.left`  
version: `0.0.4-alpha-5`  
main/style: `left.css`  

## Installation
npm:
```bash
npm install minions.left@0.0.4-alpha-5
```

browser:
```html
<link rel="stylesheet" href="https://unpkg.com/minions.left@0.0.4-alpha-5" />
```

## Code
```css
/*! minions.css */
.l-0{left:0}
.l-1p{left:1px}
.l-2p{left:1px}
.l-3p{left:3px}
.l-4p{left:4px}
.l-\.25{left:.25em}
.l-\.5{left:.5em}

.l-2{left:2em}
.l-3{left:3em}
.l-4{left:4em}

.l-2r{left:2em}
.l-3r{left:3em}
.l-4r{left:4em}

```
